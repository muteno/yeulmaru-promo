# Part 2 — Audit 발견 사항

## 🔴 Critical

### CR-1. PIN Brute Force 위험 — Rate Limit 없음

**Fact** (API ping으로 확정):
- `/api/auth/set-password` POST에 연속 5회 동일 IP 시도 → 5회 모두 차단 X
- Worker 코드: rate limit / IP 차단 / 시도 횟수 제한 모두 없음

**위험**: PIN 4자리 = 10000 조합. 평균 5000회면 매칭. 분당 무제한 시도 가능 = 약 10분이면 PIN 발견 + 계정 탈취.

**조치** (workers.dev라 WAF Rate Limiting Rules는 적용 안 됨):
1. Cloudflare 대시보드 → Workers & Pages → `yeulmaru-promo-api` → **Bindings** 탭
2. + Add → **Rate Limiting** 선택
3. Binding name: `RATE_LIMITER`, Namespace ID: `1001`, Limit: `5`, Period: `60`
4. Worker 코드 patch (set-password endpoint 시작부):

```js
if (env.RATE_LIMITER) {
  const ip = request.headers.get("CF-Connecting-IP") || "unknown";
  const { success } = await env.RATE_LIMITER.limit({ key: `setpw:${ip}` });
  if (!success) return json({ error: "너무 많은 시도. 잠시 후 다시 시도하세요." }, env, 429);
}
```

5. Quick Edit으로 재배포

## 3.2 PC 환경 자동 감지 (중요!)

**세션 도중 PC 바뀌면 path가 달라짐**:
- 회사 PC: `C:\Users\황세웅\...`
- 집 PC: `C:\Users\Hwang\...`

**해결**: 항상 `$env:USERPROFILE` 변수 사용 (PowerShell 안에서). DC `read_file`/`write_file`/`edit_block`은 literal path라 *현재 어떤 PC인지 확인 후* 명시.

확인 방법:
```powershell
Write-Host "USERPROFILE: $env:USERPROFILE"
```

## 3.3 ❌ 안 되는 패턴

```powershell
# 변수 + & invocation
$g = "C:\Program Files\Git\bin\git.exe"
& $g status    # CantActivateDocumentInPipeline 에러

# ArgumentList에 한글+공백 path
Start-Process -FilePath $g -ArgumentList @("-C", "C:\Users\황세웅\OneDrive - GS칼텍스 예울마루\...")
# → "unknown option: -" (quote escape 깨짐)

# cmd shell + 한글 path
cd /d "C:\Users\황세웅\OneDrive - GS칼텍스 예울마루\..."   # 경로 못 찾음 (인코딩)
```

## 3.4 ✅ Golden Pattern

```powershell
$p = Start-Process `
  -FilePath "C:\Program Files\Git\bin\git.exe" `      # 직접 path
  -ArgumentList @("status", "-sb") `                  # 짧고 한글 X
  -WorkingDirectory "$env:USERPROFILE\OneDrive - GS칼텍스 예울마루\DAX\Sewoong Hwang\yeulmaru-promo" `
  -NoNewWindow -Wait -PassThru `
  -RedirectStandardOutput "$env:TEMP\out.txt" `
  -RedirectStandardError "$env:TEMP\err.txt"
"EXIT: $($p.ExitCode)" | Out-File "$env:TEMP\meta.txt" -Encoding utf8
```

**핵심 5원칙**:
1. 변수 X, 직접 string
2. `-WorkingDirectory`로 작업 위치 (`-C` ArgumentList 피하기)
3. 출력은 파일 redirect → `read_file`
4. `-PassThru`로 ExitCode 잡기
5. Out-File `-Encoding utf8`

## 3.5 PowerShell cmdlet은 OK

```powershell
Select-String -Path "$wd\index.html" -Pattern "TODO|FIXME"  # ✅ stdout 잘 잡힘
Invoke-WebRequest -Uri "$base/api/health" -Headers $h        # ✅ OK
```

## 3.6 ConvertFrom-Json 한글 property

```powershell
$_.PIN          # ✅ 영문
$_.담당자       # ⚠️ empty 가능 (5.1 이슈)
$_."담당자"     # ✅ quote 묶기
```
