1분해 규칙 — 시·군·구를 한 단위로

통합·특별시는 시 자체가 1주소이고 그 아래 구/시/군이 2주소, 일반 도는 도가 1주소·시/군이 2주소. 그래서 주소2가 전국 어디든 같은 급(시군구)이 되어 지역 집계가 바로 가능하다.

  • 주소3 = 읍·면·법정동이 있으면 그것, 도로명주소라 동이 없으면 도로명(~로·~길). 104동 같은 아파트 동은 법정동으로 오인하지 않게 차단.
  • 시도 표기 통일: 전남·전라남도·광주·광주광역시 → 전남광주통합특별시 하나로. 나머지도 정식명(서울→서울특별시, 경기→경기도, 전북·전라북도→전북특별자치도 등). 표기가 몇 글자씩 어긋나던 게 전부 한 값으로 합쳐짐.
  • 우편번호 = 별도 컬럼. 신형 5자리 그대로, 구형 6자리는 값 보존+구형우편번호 플래그.

잔여 30행 = 영문 주소, ㅇㄴ·빈값 같은 정보 부재, 전국에 흔한 도로명뿐이라 특정 불가한 것 — 「주소요확인」 시트에 모아둠(주소플래그로 필터 가능).

검증: 유형별 스팟체크 + 복구 211행 전수 눈검사 + 재조합 토큰 손실 검사(샘플 3,000행 — 실질 손실 0) + 아파트동 오인 0건 + 시도 오탐 검사(아파트명 속 「세종」 오탐을 검증에서 잡아 토큰 경계 규칙으로 수정 완료).
